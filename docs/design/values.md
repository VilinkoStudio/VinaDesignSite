# 价值观
> Vina Design 在“轻量、多元、开放、感知“价值观的基础上开发体现自身特色的开源项目和产品。

## 轻量
轻量是产品体验的基础。它意味着简洁、高效、流畅，不增加用户的认知负担。我们通过优化性能、简化界面逻辑、确保产品在使用中始终保持清晰的呼吸感与掌控感。

## 多元
多元体现的是对用户价值尊重。产品应天然包容不同的使用习惯、设备条件。多元本质上理解差异，具备可扩展性，支持进行灵活的共建。

## 开放
开放是产品成长的方式。我们提供文档和可复用的资源，降低协作与创新门槛。倾听反馈，保持透明，与外部建立顺畅的连接。开放建立在信任之上，目的是让产品产生更深入的价值共鸣，实现可持续的共同创新。

## 感知
感知是体验深度的体现。产品应能理解用户的情境与意图，并给予细腻、及时的反馈。感知设计的目标是创造一种被自然理解、体贴而不过度打扰的体验。
