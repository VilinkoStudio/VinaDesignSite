## 为什么会有 Vina Design
说起来很抽象，过去 Vilinko Studio 的官方应用设计并没有统一标准，为此，我们制订了此套方案。

<br>

## 为什么选择开源
对 Vina Design 而言，设计规范是我们持续打造统一、高效设计语言的重要基础，也是实现产品体验一致性的关键支撑。

通过这份文档，我们希望能够为设计与开发团队提供清晰、可靠的设计指引，助力每一位成员更自如地构建用户喜爱的产品。

未来，我们将持续沉淀经验、优化规范，与团队一同成长，让设计在实际项目中创造更大价值。

<br>

## 设计指南
- Values 价值观
- Color 色彩
- Controls 控件
- Layout 布局
- Motion 动效
- Dark Mode 深色模式
